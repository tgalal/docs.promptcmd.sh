# Uninstall & Cleanup

::: info
In addition to the below info, you may also want to delete the configuration
files and prompts, see the paths listed under [Lookup
Paths](/getting-started/lookup-paths).
:::

## Linux/macOS (Shell-based Installation)

```bash
rm -rf ~/.promptcmd/bin/promptcmd ~/.promptcmd/bin/promptctl
```

## Homebrew

```bash
brew uninstall promptcmd
```

Optionally remove configuration and data:

```bash
rm -rf ~/Library/Application\ Support/promptcmd
```

## Windows (MSI)

Use the "Add or Remove Programs" settings to uninstall promptcmd.
